AlgoGauge

This Project was made in conjuction with Blake Thomas, Trace Ericson, and Hunter Ferrin. The link to the project. https://github.com/19terickson/AlgoGaugeCapstone.git

We used the MERN stack to create this Website. The Goal of this Website is to allow the user to check the speed of two algorithms of their chosing. The main points I worked on, implimenting a node module highlight to make the code look beautiful on the front end. I also implimented the server functionality to call the queue and our test function to remove from the queue and fake run the algorithms and then push that fake data into the Mongo Database.

Queue

This is the Queue Modal, we have it on a 30 second timer to pull people out of the queue then push fake data into the results Mongo database. Once it registers that place in queue is -1 the results button will pop up and when clicked it will pull from the results database.

Metrics Page

The metrics page shows all prior runs from all users and then we have a your metrics page that pulls prior runs based on the username that was stored in local storage.
